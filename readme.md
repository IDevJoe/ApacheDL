# ApacheDL
Download files from improperly configured Apache webservers.

```
ApacheDL -u [url]
  -d, --debug                Show debug messages
  -u, --url=VALUE            Specify the URL to download from
  -o, --output=VALUE         Specify the output directory, defaults to 'out'
  -h, --help                 Show this
```
